Cursor Test Page

Hover over each zone to preview the cursor. Linux file names shown in blue. Descriptions explain where you'd see each cursor in real use.

Core

Normal Select cursor: default linux: default Desktop, buttons, most UI elements
Text Select cursor: text linux: text (alias: xterm, ibeam) Input fields, text editors, terminals
Hand / Link cursor: pointer linux: pointer (alias: hand2, hand1) Links, clickable buttons, interactive elements
Busy cursor: wait linux: wait (alias: watch) System fully busy, cannot interact
Working in Background cursor: progress linux: progress (alias: left_ptr_watch) App loading but you can still click elsewhere
Precision Select cursor: crosshair linux: crosshair (alias: cross, tcross) Drawing apps, screenshot region select, color pickers
Help Select cursor: help linux: help (alias: question_arrow) "What's this?" mode, help tooltips
Unavailable cursor: not-allowed linux: not-allowed (alias: forbidden, crossed_circle) Disabled buttons, invalid drop targets
Context Menu cursor: context-menu linux: context-menu Right-click menu available

Resize & Move

Move (Cross) cursor: move linux: size_all (alias: move, fleur, all-scroll) Dragging a file icon, moving a window by title bar
Resize Vertical cursor: ns-resize linux: ns-resize (alias: v_double_arrow, size-ver) Top/bottom window borders, vertical split handles
Resize Horizontal cursor: ew-resize linux: ew-resize (alias: h_double_arrow, size-hor) Left/right window borders, horizontal split handles
Resize NW-SE (diagonal \) cursor: nwse-resize linux: nwse-resize (alias: bd_double_arrow, size_fdiag) Top-left or bottom-right corner of a window
Resize NE-SW (diagonal /) cursor: nesw-resize linux: nesw-resize (alias: fd_double_arrow, size_bdiag) Top-right or bottom-left corner of a window
Column Resize cursor: col-resize linux: col-resize Dragging a column divider in tables or split panes
Row Resize cursor: row-resize linux: row-resize Dragging a row divider in tables or split panes
Resize North cursor: n-resize linux: n-resize (alias: top_side) Dragging the top edge of a window
Resize South cursor: s-resize linux: s-resize (alias: bottom_side) Dragging the bottom edge of a window
Resize East cursor: e-resize linux: e-resize (alias: right_side) Dragging the right edge of a window
Resize West cursor: w-resize linux: w-resize (alias: left_side) Dragging the left edge of a window
All Scroll cursor: all-scroll linux: all-scroll (alias of size_all) Middle-click scroll mode in browsers

Grab & Drag

Open Hand (Grab) cursor: grab linux: openhand (alias: grab) Pannable canvas, Google Maps hover, PDF viewer hover
Closed Hand (Grabbing) cursor: grabbing linux: closedhand (alias: grabbing) Actively dragging a map, panning a canvas
Copy (Drag) cursor: copy linux: copy (alias: dnd-copy) Ctrl+drag to copy a file
Alias / Shortcut cursor: alias linux: alias (alias: link, dnd-link) Creating a symbolic link / shortcut via drag
No Drop cursor: no-drop linux: dnd-none (alias: dnd-no-drop) Dragging over a target that won't accept the drop

Specialty

Zoom In cursor: zoom-in linux: zoom-in Image viewers, map zoom in
Zoom Out cursor: zoom-out linux: zoom-out Image viewers, map zoom out
Cell cursor: cell linux: cell (alias: plus) Spreadsheet cell selection
Vertical Text cursor: vertical-text linux: vertical-text Vertically-oriented text fields
None (hidden) cursor: none linux: (no file needed) Video playback, presentations, kiosk mode

Linux-only (no CSS equivalent, cannot preview here)

Pencil / Draw linux: pencil (alias: pen) Drawing/annotation mode in image editors
Color Picker linux: color-picker Eyedropper tool picking a color from the screen
Split Horizontal linux: split_h Hovering over a horizontal split handle
Split Vertical linux: split_v Hovering over a vertical split handle
Up Arrow linux: up-arrow Selection cursor pointing up
Center Pointer linux: center_ptr Arrow pointing up-center (some WMs use for root window)
Alternate Select (Right Ptr) linux: right_ptr Mirrored arrow — some WMs use for inactive windows or root window
DnD Move linux: dnd-move Dragging to move an item (no copy/link modifier)
X Cursor linux: X_cursor Fallback/error cursor (large X shape)
